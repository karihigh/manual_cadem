MANUAL DE NORMAS PARA LA CONSTRUCCIÓN
DE UN REPORTE WEB CADEM

Guías para su desarrollo y recomendaciones gráficas

La construcción efectiva de un reporte web nace a partir de un proceso de recolección de la información y de comprensión de la misma para luego comunicar de manera intencionada los aspectos más relevantes a un cliente dado.

Este manual tiene por objetivo guiar el desarrollo de los reportes web construidos por Cadem, poniendo especial énfasis en la implementación de la biblioteca de gráficos interactivos diseñados especialmente para estos reportes.

Se propone que el proceso para el desarrollo de los reportes se realice en cinco grandes etapas que se detallan a continuación.
